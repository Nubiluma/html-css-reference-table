HTML & CSS References

HTML

Element Tag Element Name Description/Uses
a anchor Used for creating hyperlinks to websites, files, ...
p paragraph Visualizes blocks of text
h1-h6 section heading Headings in HTML document (descending hierarchy)
ul unordered list Typically renders list items as a bulleted list
ol ordered list List items will be listed with ascending numbers instead of bullet points
li list item Used to present an item in a list or menu
div content division A pure, generic container used to group content (great for CSS styling)
body document body Presents the content of an HTML document (Only one body per document)
br line break Produces a line break in text (useful for poems or addresses)
img image embed Embeds an image into an HTML document
link external resource link Links current document to an external resource (e.g. stylesheets)
title document title Defines the documents title shown in browser's title bar or page's tab
table table Presents content in a table (needs tr and td elements which contain the actual content/data)
thead table head Defines a set of rows as head of the columns of a table
tbody table body Encapsulates a set of table rows of a table
tr table row Defines a row of cells in a table
th table header Defines a cell as header of a group of table cells
td table data cell Defines a cell of a table that contains data
article article contents
section generic section
blockquote block quotation Indicates that the enclosed text is an extended quotation
q inline quotation Encloses text with quotation marks
strong strong importance Indicates that its contents have strong importance/seriousness/urgency. (Important for screen readers) It typically renders text content in bold type by default
em emphasis Marks text to be emphasized (Useful for screen readers). It is usually shown as italic letters and can be nested indicating greater degrees of emphasis
small side comment Represents side-comments (like small/fine print). Typically renders text within one font-size smaller
span content span Generic inline container only used for styling purposes (like the div element)
